# 🌦️ Weather App

A modern and responsive Weather App built using **HTML, CSS, and JavaScript**. The application provides a clean and interactive interface where users can search for a city and view weather information in an easy-to-read format.

The project focuses on creating a responsive user experience with a well-structured layout, smooth interactions, and an attractive design. It demonstrates the use of JavaScript for handling user input, updating content dynamically, and improving overall interactivity.

## ✨ Features

* 🌍 Search weather by city name
* 🌡️ Display weather details in a clean layout
* 📱 Fully responsive design
* ⚡ Dynamic content updates using JavaScript
* 🎨 Modern and user-friendly interface
* 💻 Beginner-friendly frontend project

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

This project was created to practice frontend web development concepts, including responsive design, DOM manipulation, and JavaScript-based interactivity while building a clean and functional weather application.
